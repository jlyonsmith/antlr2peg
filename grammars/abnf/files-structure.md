The *abnf* directory has the following structure/files:
  - Abnf.g4: ANTLR grammar
  - Abnf.lua: PEG parser generated by antlr2peg from Abnf.g4. It is almost a direct translation.
  - Abnf-auto.lua: PEG parser generated by pegparser from Abnf.lua by rewriting non-disjoint repetitions, lexical rules, etc.
  - examples/: directory with examples used to validate Abnf.g4
  - grammarinator/: directory with tests/files generated by grammarinator
  - outputNoTk/: directory with examples generated from examples/ by randomly removing a token
  - nonLiteralRewriting.ods: spreadsheet with the analysis of the non-literal rewritings
  - howtorun.md: file indicating how to run/generate the files.
